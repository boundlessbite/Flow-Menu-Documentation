Overview

Flow Menu adds an elegant, lightning-fast quick-add popup directly to the Unreal Engine Level Viewport. It allows world artists, level designers, and developers to easily spawn commonly used items instantly without interrupting their viewport workflow or searching through the Content Browser.

Specification Details
Plugin Name Flow Menu
Category Editor Utility / Quality of Life
Supported Asset Types Actor classes, Actor Blueprint assets, static mesh assets, and lights
Primary Workflow Open popup with a shortcut, choose an item, then place it directly at the cursor in the viewport.

Editor-Only Guardrail: Flow Menu is compiled strictly for editor modules. It will not package into your runtime games and will never impact your game's final shipping UI or size.

Key Features

Quick-Add Popup

Opens directly inside the Level Viewport contextually at your mouse pointer location.

Custom Shortcuts

Assign your preferred hotkeys smoothly through Unreal's standard Project Settings page.

Multi-Class Spawning

Seamless support for placing native classes, Blueprint Actors, custom lights, or static meshes.

Keyboard Navigation

Move quickly using arrow keys and press Enter to place, keeping your hands on input tools.

Deferred Placement Mode

Enabling click-to-place lets you select the item, inspect its proxy, and tap to drop in real-time.

Pre-configured Defaults

Ships with rich default options that can be edited, overwritten, or supplemented on a per-project basis.

Installation

Install Flow Menu as a global project plugin or distribute it as a localized plugin folder embedded with your production directory structure.

1

Copy Folder

Copy the Flow Menu source folder directly into your current Unreal project's Plugins directory.

2

Open Project

Launch the Unreal Engine 5 project in your targeted version.

3

Enable the Plugin

Navigate to Edit > Plugins. Search for "Flow Menu", and check the active box to enable it.

4

Restart the Editor

If prompted, click the restart button on the bottom right corner of the editor window to reload modules.

Basic Usage

Once installed, using Flow Menu takes less than a second during active level design runs:

  • Open Level Viewport: Focus on any 3D level viewport within the editor.
  • Call Shortcut: Click the pre-configured hotkey shortcut to open the Flow Menu immediately.
  • Select Entry: Navigate choices with the mouse pointer or step list selections using the key arrow bindings and hit Enter.
  • Placement Spawning: The selected actor will spawn right under the viewport mouse trajectory, or anchor with real-world clicks depending on the configuration.

Where To Configure It

You can adjust shortcuts, default spawners, alignment filters, and custom entry maps inside Project Settings:

Settings Configuration Path Edit > Project Settings > Plugins > Flow Menu
Setting Purpose
Open Menu Shortcut Defines the specific key combination mapping used to open the popup interface in the Level Viewport.
Placement Mode Controls whether the item spawns instantly at the raycast hit, or delays spawning until you physically click on the scene.
Align To Surface Normal Orients the spawned actor to correspond directly to the slope angle of the hit surface underneath.
Default Entries Contains the sample out-of-the-box templates shipped with the plugin (lights, primitives). These items can be expanded, changed, or completely cleared.
Custom Entries Project-defined lists for custom Actor classes, specific Blueprint files, or heavy mesh components you wish to catalog.

Supported Menu Entries

Flow Menu dynamically handles several core categories of assets directly in one unified user menu:

Actor Class

Highly practical for placing default engine primitive shapes, utilities, spawners, triggers, volumetric boxes, and core structural gameplay nodes.

Actor Blueprint

Perfect for complex, customized project files like interactive doors, gates, player cameras, lighting rigs, audio cues, or environment assets containing gameplay logic.

Static Mesh

Optimized for lighting-fast decoration and greyboxing. Pop common modular wall blocks, pillars, kitbash pieces, or obstacles into active play spaces instantly.

Placement Behavior

To maintain stability and predictable asset orientation, Flow Menu operates under precise rules calculated sequentially based on the Level Viewport's camera state:

  • Level Viewport Focus: The popup opens exclusively if triggered while the level editor viewport is currently selected and active.
  • Collision Raycast Tracing: The editor casts a line trace under your cursor. If the ray hits geometric actors (landscapes, meshes, BSP), the item spawns directly at that specific hit coordinate.
  • Fallback Vector Placement: If the mouse pointer is hovered over empty skies or void spaces where no collision geometric is hit, the spawn coordinates fall back to a configured distance vector in front of the active editing camera.
  • Orientation Preservation: Static mesh models and custom Actor Blueprints preserve their precise authored default rotation variables unless "Align To Surface Normal" overrides it.
  • Align to Surface Normal: When active, this feature dynamically calculates the hit surface's angle vector and matches the upward z-rotation of the spawned actor to blend flush against complex terrain curves or sloped architectural meshes.

Keyboard Workflow

For designers who prefer keyboard input to maintain maximum speed, Flow Menu includes deep shortcut tracking options:

Navigation
Up Down

Steps upward or downward through the active list. Highlight cycles sequentially.

Accept Selection
Enter

Confirms the highlighted option and instantiates the spawning sequence inside the viewport.

Dismiss Popup
Esc

Instantly closes the quick menu without choosing an item or altering any world geometry.

Pro-Tip: The pop-up container supports automatic content scrolling. If your active list size extends beyond the height limits of the viewport screen, keyboard scrolling triggers smoothly so no options are blocked.

Troubleshooting & Support

The popup doesn't open.

Ensure the Level Viewport is explicitly active before triggering the hotkeys. Click anywhere in the viewport viewport area to draw focus first, then tap the key sequence.

An item from the menu fails to spawn.

Verify that the selected entry correctly references a valid Actor class, a real, uncorrupted Actor Blueprint asset, or a properly-referenced static mesh. If references break inside the Content Browser, Flow Menu will safely skip creation to avoid crashing the editor.

The hotkeys conflict with another editor shortcut.

Navigate immediately to Project Settings and bind an unused key combination to the Open Menu Shortcut entry mapping.

Code changes to the plugin are not updating.

If you are making modifications or manual updates to the plugin files on disk, always restart the Unreal Editor to clean up modules and reload the compiled binaries seamlessly.
